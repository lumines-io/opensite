# Admin Flow

## Overview

This document describes the user experience for administrators who have full access to manage the HCMC Road Construction Tracker, including system configuration, user management, and content administration.

## Prerequisites

- Registered account with verified email
- Role: `admin`

## Admin Capabilities

| Area | Capabilities |
|------|--------------|
| Content | Full CRUD on all collections |
| Users | Create, edit, delete, assign roles |
| Moderation | All moderator abilities |
| System | Feature flags, scrapers, analytics |
| Database | View logs, manage data |

## Admin API Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/admin/users` | GET | List all users |
| `/api/admin/users/[id]` | PATCH | Update user |
| `/api/admin/suggestions` | GET | List all suggestions |
| `/api/admin/feature-flags` | GET/POST | Manage flags |
| `/api/admin/scrapers/[id]/run` | POST | Trigger scraper |
| `/api/admin/cache` | POST | Clear cache |
| `/api/admin/analytics` | GET | Get analytics |

## Security Considerations

### Admin-Only Actions

- User role changes
- Feature flag modifications
- Scraper manual triggers
- Cache clearing
- Data exports
- System configuration

### Audit Trail

All admin actions are logged:
- Who performed the action
- What was changed
- When it happened
- Previous/new values
