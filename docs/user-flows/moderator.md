# Moderator Flow

## Overview

This document describes the user experience for moderators who review and manage community suggestions on the HCMC Road Construction Tracker.

## Prerequisites

- Registered account with verified email
- Role: `moderator` or `admin`
- Feature flag: `FEATURE_MODERATOR_DASHBOARD` enabled

## Moderator Responsibilities

### Community Suggestions
1. Review pending suggestions
2. Verify evidence and accuracy
3. Approve, reject, or request changes
4. Merge approved suggestions into database
5. Maintain data quality

### Private Construction Review
1. Review submissions from Sponsors/Developers
2. Verify marketing content accuracy
3. Check location/geometry data
4. Ensure compliance with platform policies
5. Approve and publish to map
